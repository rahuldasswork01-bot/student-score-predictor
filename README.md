# 🎯 Student Score Predictor

A Machine Learning project that predicts a student's exam score based on hours studied.

## 📊 Project Overview
This project uses Linear Regression to predict exam scores.
Given the number of hours a student studies, the model predicts their expected exam score.

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Matplotlib
- Scikit-learn (Linear Regression)
- Jupyter Notebook

## 📈 Results
- Model Accuracy: R² Score = 1.00
- Predicted score for 6.5 hours of study = 73.8

## 🔍 What This Project Does
1. Creates a student dataset
2. Visualizes data with scatter plot
3. Trains a Linear Regression ML model
4. Predicts scores for new students
5. Draws prediction line on graph
